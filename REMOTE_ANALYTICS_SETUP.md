# 전체 방문자 통계 연결 가이드

GitHub Pages는 정적 호스팅이라 방문자 데이터를 직접 저장할 수 없습니다.  
다른 사용자 접속통계까지 보려면 중앙 저장소가 필요합니다.

## Google Apps Script로 연결

1. Google Drive에서 새 Google Spreadsheet를 만듭니다.
2. 메뉴에서 `확장 프로그램 > Apps Script`를 엽니다.
3. 이 저장소의 `google-apps-script.js` 내용을 붙여넣습니다.
4. `배포 > 새 배포 > 웹 앱`을 선택합니다.
5. 실행 권한은 본인, 액세스 권한은 `모든 사용자`로 설정합니다.
6. 발급된 웹앱 URL을 복사합니다.
7. `config.js`의 값을 아래처럼 수정합니다.

```js
window.MEDIS_REMOTE_ANALYTICS_URL = "여기에_웹앱_URL";
```

수정 후 GitHub에 다시 올리면 `admin.html`에서 모든 방문자 통계를 볼 수 있습니다.

## 수집되는 정보

- 접속 시간
- 페이지/이벤트 종류
- 시간대·언어 기반 지역 추정
- 브라우저, OS, 기기 유형
- 화면 크기, 뷰포트
- CPU 코어, 메모리, 네트워크 타입
- 유입 경로

정확한 GPS 위치, 이름, 전화번호, 주민번호 같은 민감정보는 수집하지 않습니다.
